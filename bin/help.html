Updating

Upon startup, github is checked to see if there is a newer version available, if so, you are prompted to see if you want this new version. It might take a few minutes to download and the install started.

Optionally you can go to Help -> Upgrade and will be presented with:

  • Version of the currently running program
  • Version of the code on github, this will likely be at least as new as your running program
  • A list of installable versions found on github. These will likely include some older versions and something close to the code version on github, but might be a bit older because an installable version does not have to be generated everytime the code is updated on github
You can click on an installable version, and after some time, the download will start and the install started.
